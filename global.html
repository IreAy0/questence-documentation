# ActivityIndicator(visible)

A loading indicator component that displays an animated spinner when visible is true. By default, the loading indicator is not visible.
Parameters:
Name Type Description
visible boolean Whether the loading indicator should be visible or not. Default is false.

View Source components/ActivityIndicator.jsx, line 32

Example
import React from 'react';
import ActivityIndicator from './ActivityIndicator';

function MyComponent() {
  const [loading, setLoading] = React.useState(false);

  function handleClick() {
    setLoading(true);
    // Simulate a delay with setTimeout
    setTimeout(() => setLoading(false), 2000);
  }

  return (
    <div>
      <button onClick={handleClick}>Click me!</button>
      <ActivityIndicator visible={loading} />
    </div>
  );
}

# App() → {JSX.Element}

App Component This component manages the navigation and state for an LMS or Authoring platform. It dynamically handles the URL patterns for both LMS (with `app_calling = 12`) and Authoring platform (with `app_calling = 13`). Based on the detected calling application, it fetches data such as course content, comments, and bookmarks/notes. The component is also responsive, adjusting between desktop and mobile views. - LMS URL format: `{lms-url}/12/{course_id}/{enrollment_id}/{page_id}/{token}?redirect_url={redirect_url}` - Authoring URL format: `{lms-url}/13/{course_id}/{page_id}/{token}?redirect_url={redirect_url}` The component saves key data (e.g., `lms_token`, `course_id`, `enrollment_id`, and `redirect_url`) in `localStorage` and handles responsive behavior for desktop and mobile devices.
